Design a complete enterprise-grade Masjid ERP UI/UX system in Figma.

This product must match:
- Zoho-level polish
- Tally-level financial clarity
- ERPNext-level modular structure

Product Type:
A multi-tenant mosque ERP system managing members, households, donations, funds, accounting, events, announcements, staff, assets, and reports.

Design Requirements:
- Desktop-first ERP with mobile-responsive layouts
- Use 8px spacing system
- Minimum touch target: 48px
- Grid: 12-column layout (max width 1440px)
- Sidebar width: 240px (collapsible)
- Topbar height: 64px
- Table row height: 48–56px
- Card padding: 16–24px
- Border radius: 8–12px

Design Style:
- Clean enterprise UI (NOT startup style)
- Calm, trustworthy, professional
- Use green/teal/blue primary palette
- Neutral backgrounds (#F5F7FA style)
- Subtle shadows (elevation system)
- No decorative overload

Typography:
- Strong hierarchy:
  - Page title (24–28px)
  - Section title (18–20px)
  - Body (14–16px)
  - Caption (12px)
- Financial numbers should be bold and prominent

Core Components:
- Sidebar navigation
- Topbar (search, notifications, profile)
- KPI cards
- Data tables (sorting, filtering, pagination)
- Drawer panels (right-side)
- Modal forms
- Tabs
- Status chips
- Timeline / activity feed
- Charts (bar, line, donut)
- Receipt preview UI
- Role/permission matrix
- Calendar view
- Notification center

States:
- Loading skeleton
- Empty state with CTA
- Error state
- Success confirmation
- Disabled states
- Permission restricted states

Interactions:
- Table row → opens detail drawer
- Primary CTA always visible
- Forms open in modal/drawer
- Confirmations for finance actions
- Toast notifications after actions
- Real-time updates for dashboard

Deliver:
- Design system (colors, typography, spacing, components)
- Full desktop screens
- Mobile responsive screens
- Interactive prototype flows

Do NOT create marketing UI. This must feel like a serious ERP used daily by mosque admins and accountants.
